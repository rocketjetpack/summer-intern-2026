# 2026 Summer Network Engineering Internship

This repo holds the curriculum site for the 2026 Summer Network Engineering Internship — a week-by-week guide covering objectives, resources, hands-on labs, and notes. It's built with [Observable Framework](https://observablehq.com/framework/).

## Running it locally

Install dependencies, then start the local preview server:

```sh
npm install
npm run dev
```

Visit <http://localhost:3000> to view the site.

## Project structure

- `src/` — site source; each Markdown file is a page (e.g. `src/week_02/objectives.md`)
- `src/week_NN/` — per-week content: objectives, resources, labs, and personal notes
- `observablehq.config.js` — site config, including the sidebar navigation
